## Run

1. Install Hyperledger Fabric latest docker images and binaries.
    ```shell
    ./install-fabric.sh
    ```
1. Generate the network artifacts.
    ```shell
    cd network-files
    ./generate.sh
    ```
1. Deploy the network.
    ```shell
    ./start.sh
    ```
